Get Started in 60 Seconds

1

Install AgentGraph

pip install agentgraph[server]
2

Start the Server

agentgraph-server

Dashboard at http://localhost:8080

3

Track Your Agents

from agentgraph import AgentGraphClient

client = AgentGraphClient(api_key="...")

# Log events
client.log("tool.call", action="search", input_data={...})

# Query history
result = client.query("what happened to customer X?")
